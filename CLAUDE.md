# Amanda Gráficos

Sitio web de Amanda Gráficos, negocio de diseño personalizado (stickers, candy bar, vinilos, tarjetería, etc.) en Rosario, Argentina. Activo desde 2018.

## Archivos principales

- `index.html` — página principal (hero, packs, precios, contacto)
- `catalogo.html` — catálogo de stickers
- `trabajos.html` — galería de trabajos realizados
- `app.js` — lógica del catálogo (carrito, envío por WhatsApp)
- `estilos.css` — todos los estilos

## Deploy

El sitio vive en [www.amandagraficos.store](http://www.amandagraficos.store) y se publica vía GitHub Pages:

```
git push origin main
```
